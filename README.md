
# Library Management System

A simple Library Management System developed using Python. This project is designed to manage library records using file handling and a menu-driven approach. Users can add books, remove books, display all available books, and search books using Book ID.

The project stores data permanently in a CSV file, making it easy to maintain library records even after closing the program.

---

## Features

* Add New Book
* Remove Existing Book
* Display All Books
* Search Book by Book ID
* CSV File Data Storage
* Duplicate Book Checking
* Menu Driven Console Application

## CSV File Format

The system stores book records in the following format:

```bash
Book_ID,Book_Name,Author,Publish_Year
```

Example:

```bash
101,Python Basic,Saurabha Srivastava,2027
```
